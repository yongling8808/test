Position规则

a.当前为<p>标签，设置了禁止换行style，该行文本在正常加载及双指缩放的情况下都不会发生折行！

b. CSS Position:static,可折行排版。标签在浏览网页时，由于页面中的文本较小，需要进行放大操作，但是在放大页面后，原本同一行内的文本超出屏幕范围，在阅读期间需要水平方向拖动页面来查看，尤其对于文本行数较多的页面来说，需要反复进行左右滑动的操作，较为不便。

c. CSS Position:relative,可折行排版。标签在浏览网页时，由于页面中的文本较小，需要进行放大操作，但是在放大页面后，原本同一行内的文本超出屏幕范围，在阅读期间需要水平方向拖动页面来查看，尤其对于文本行数较多的页面来说，需要反复进行左右滑动的操作，较为不便。

d. CSS Position:absolute,不可折行排版。标签在浏览网页时，由于页面中的文本较小，需要进行放大操作，但是在放大页面后，原本同一行内的文本超出屏幕范围，在阅读期间需要水平方向拖动页面来查看，尤其对于文本行数较多的页面来说，需要反复进行左右滑动的操作，较为不便。

e. CSS Position:fixed,不可折行排版。标签在浏览网页时，由于页面中的文本较小，需要进行放大操作，但是在放大页面后，原本同一行内的文本超出屏幕范围，在阅读期间需要水平方向拖动页面来查看，尤其对于文本行数较多的页面来说，需要反复进行左右滑动的操作，较为不便。
